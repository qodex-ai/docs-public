---
title: "Selecting an Existing Collection"
---

### If you’ve already imported collections:

1. Go to **AI Agent**
2. Top-left dropdown → select existing collection

### Alternate way:

- Go to **Test Scenarios** → left sidebar → **Collections**
- Select and reuse any existing collection for new test scenarios

### Why:

- Reuse existing API collections for:
  - New functional test scenarios
  - New OWASP test runs
  - Additional environments (staging, prod)
- Maintain multiple API versions (v1, v2) side by side
