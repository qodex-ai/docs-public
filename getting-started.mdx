title: "Get Started in 10 Minutes"
---

Here’s how to start testing your APIs with [Qodex.ai](http://app.qodex.ai) — no manual scripts required:

### 1. Sign Up

Go to [app.qodex.ai](http://app.qodex.ai) → Create your account → You’ll land in **AI Agent**.

![Screenshot2025 06 25at10 59 17 Pn](/images/Screenshot2025-06-25at10.59.17.png)

### 2. Import or Discover your APIs

\
In **AI Agent** → top-left dropdown → **Add Collection**:

- Upload swagger.json, swagger.yaml, or Postman collection
- OR auto-discover APIs with Qodex SDK:

  ```bash
  curl -sSL https://raw.githubusercontent.com/qodex-ai/swagger-bot/refs/heads/main/run.sh -o script.sh && chmod +x script.sh && ./script.sh --project-api-key YOUR_PROJECT_KEY
  ```

For more details: [**API Discovery Agent**](https://docs.qodex.ai/api-discovery-agent)

![Add Collection Gi](/images/Add-Collection.gif)

### 3. Generate Test Scenarios in Chat

In **AI Agent chat** → Type your test prompt:

**Examples:**

- _Test login flow with valid and invalid credentials_
- _Run OWASP Top 10 tests on my APIs_
- _Check token expiration and re-authentication flow_

**Qodex will generate:**

- Test Scenario
- Test Cases within test scenario
- Request Data
- Test Rules
- API chaining (if needed)

For more details: [**Test Scenario Creation Agent**](https://docs.qodex.ai/test-scenario-creation-agent)

![Generate Ts Ai Agent Gi](/images/generate-ts-ai-agent.gif)

### 4. Run Tests

After saving → Go to **Test Scenarios** (top nav).

From here:

- Run a single scenario → Click **Run**

You’ll see results in **Build** panel on the right as tests run.

For more details: [**Running Tests from the Qodex Web App**](https://docs.qodex.ai/running-tests-from-qodex-web-app)

![Screenshot2025 06 25at11 06 20 Pn](/images/Screenshot2025-06-25at11.06.20.png)

### 5. View Results

Once test run completes → Go to \*\*Build \*\*panel -\> Expand -\> **View Detail**:

- View pass/fail
- See request/response
- Drill into any failed test
- If a test failed due to API changes → Click **Fix Me** → Auto-update → Re-run

For more details: [**Build Reports**](https://docs.qodex.ai/build-reports)
