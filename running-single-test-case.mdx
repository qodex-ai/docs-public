---
title: "Running Single Test Case"
---

Qodex lets you run an individual test case — useful when debugging or working on a single endpoint.

## Steps:

1. Go to **Test Scenarios**
2. Open a Test Scenario → Select a single test case
3. Click **Run**

## When to use:

- Debugging a failing test
- Checking a new test case after creation
- Isolating edge cases
